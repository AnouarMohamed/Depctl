# Depctl v1.1.0

## Summary

This release introduces provider-aware deployment planning and the first autopilot targets.

## Highlights

- Adds target providers for VPS, Vercel, and Fly.io.
- Adds short `setup` and `deploy` flows.
- Adds plan v0.2 with target metadata, artifacts, checks, credentials, secret imports, actions, and rollback metadata.
- Adds root artifact backups before overwrites.
- Adds provider-safe `.env` import without storing secret values in generated plans or reports.
- Improves generated Docker, Compose, and provider templates.
- Adds CI quality gates and release binary publishing.

## Validation

- `make verify`
- Manual provider E2E validation is still required before marking this release final.
