# Depctl v1.1.2

## Summary

This release fixes the Go install path so users get a lowercase `depctl` command.

## Changes

- Adds `github.com/AnouarMohamed/Depctl/cmd/depctl` as the official installable command package.
- Updates install docs to use:

  ```bash
  go install github.com/AnouarMohamed/Depctl/cmd/depctl@latest
  ```

- Documents adding `$(go env GOPATH)/bin` to `PATH` when the shell cannot find Go-installed binaries.
- Moves the embedded startup banner into a shared internal package so both entrypoints use the same banner.
- Updates local and container builds to use the official command package.
- Adds a test to ensure the root `HHHQ` banner and embedded banner stay in sync.

## Validation

```bash
make verify
```
